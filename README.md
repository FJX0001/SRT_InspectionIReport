# SRT字幕检查工具

一个功能强大的在线SRT字幕文件检查和对比工具，支持错误检测、格式验证、时间线分析和字幕对比等功能。

## 🌟 主要功能

### 1. **字幕检查功能**
- **格式验证**：检测SRT文件是否符合标准格式
- **时间码检查**：验证时间码格式（HH:MM:SS,mmm）和有效性
- **序号连续性检查**：检查字幕序号是否连续递增
- **时间重叠检测**：检测字幕时间是否重叠
- **文本长度检查**：检查单行文本是否过长（超过40字符）
- **行数检查**：检查每个字幕块是否超过2行
- **时间间隔检查**：检查字幕间隔是否过短（<100ms）

### 2. **字幕对比功能**（新增）
- **时间轴精确对比**：毫秒级时间轴差异检测
- **序号一致性对比**：检查两个字幕文件的序号是否完全匹配
- **字幕数量对比**：验证两个字幕文件的数量是否一致
- **详细差异报告**：显示每个差异的具体位置和数值
- **支持多种输入方式**：文件上传、拖放、直接粘贴

### 3. **实用工具**
- **自动修复建议**：针对检测到的问题提供修复建议
- **导出修正版**：自动修复序号不连续和格式问题并导出
- **时间线可视化**：图形化显示字幕在时间轴上的分布
- **快捷键支持**：提高工作效率

## 🚀 快速开始

### 使用字幕检查功能
1. **上传文件**：点击"选择文件"按钮或拖放SRT文件到上传区域
2. **粘贴内容**：或直接在文本框中粘贴SRT字幕内容
3. **点击检查**：点击"检查字幕"按钮开始分析
4. **查看结果**：在右侧面板查看检查结果和统计信息
5. **修复问题**：根据建议修复问题或导出修正版

### 使用字幕对比功能
1. **加载两个字幕**：在对比区域分别加载两个字幕文件
2. **开始对比**：点击"开始对比"按钮
3. **查看差异**：查看时间轴差异和序号差异统计
4. **分析结果**：根据详细报告调整字幕文件

## ⌨️ 键盘快捷键

- `Ctrl + Enter`：检查字幕

2

## 📊 检测项目说明

### 错误级别（必须修复）
- ❌ **时间码格式错误**：不符合HH:MM:SS,mmm格式
- ❌ **时间顺序错误**：结束时间早于开始时间
- ❌ **时间重叠**：两个字幕时间有重叠
- ❌ **序号格式错误**：序号无法解析为数字

### 警告级别（建议优化）
- ⚠️ **文本过长**：单行超过40个字符
- ⚠️ **行数过多**：单个字幕超过2行
- ⚠️ **序号不连续**：序号不连续递增
- ⚠️ **间隔过短**：字幕间隔小于100ms
- ⚠️ **空字幕**：字幕文本为空

## 🎯 对比功能精度

- **时间轴精度**：毫秒级（1ms）对比
- **序号对比**：完全匹配，无误差允许
- **数量对比**：字幕数量必须一致
- **位置对比**：按位置顺序对比，不考虑序号顺序
